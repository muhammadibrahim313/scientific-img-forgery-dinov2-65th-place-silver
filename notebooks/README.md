# Notebooks

The executable notebooks for this project live on Kaggle so they run with the data and weights already attached.

- **Training:** https://www.kaggle.com/code/ibrahimqasimi/train-scientific-img-forgery-dinov2-65th-place
- **Inference:** https://www.kaggle.com/code/ibrahimqasimi/infer-scientific-img-forgery-dinov2-65th-place

If you want local copies in this repo, download them from Kaggle (File → Download Notebook) and drop the `.ipynb` files here. They are not committed by default because Kaggle is the canonical source and re-running locally requires the competition data to be downloaded separately.
